# Dataset Folder

Put your labeled image dataset here.

Required format:

```text
data/images/
  class_1/
    image1.jpg
    image2.jpg
  class_2/
    image1.jpg
    image2.jpg
```

Example:

```text
data/images/
  cats/
    cat1.jpg
  dogs/
    dog1.jpg
```

Do not upload very large datasets to GitHub unless required by your instructor.
